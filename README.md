# 🎵 SoongTunes

**SoongTunes**는 Flask 기반으로 개발한 음악 추천/관리 웹 프로젝트입니다.  
사용자가 음악을 탐색하고, 취향에 맞는 곡을 찾을 수 있도록 구성한 웹 애플리케이션입니다.

---

## 📌 Project Overview

이 프로젝트는 Python과 Flask를 활용하여 웹 애플리케이션의 기본 구조를 직접 구현하고,  
음악 관련 데이터를 기반으로 사용자 친화적인 서비스를 만드는 것을 목표로 제작되었습니다.

### 주요 목표
- Flask 기반 웹 서비스 구조 이해
- 라우팅 및 템플릿 렌더링 구현
- 음악 관련 기능을 웹에서 제공
- Git/GitHub를 활용한 프로젝트 버전 관리

---

## 🛠 Tech Stack
